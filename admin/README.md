# Pulse Link Admin

Vue 3 + Tailwind hospital command dashboard.

## Setup

```bash
npm install
copy .env.example .env
npm run dev
```

The dashboard expects the Laravel backend at `VITE_API_BASE_URL`.

## Features

- Dynamic stat counters.
- SOS activation modal.
- Live tracking map with Leaflet.
- Laravel Echo + Reverb websocket updates.
- Alert and commitment timeline.
